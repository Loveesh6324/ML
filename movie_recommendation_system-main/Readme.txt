How To Run?

Step 1:
Open Jupyter notebook or collab and install 
-numpy
-pandas
-ast
-nltk
-pickle

Step 2: 
Run "movie-recommendation-system.ipynb" at the end it will give you 2 files "movies_dict.pkl" and "similarity.pkl".

Step 3:
open a python supported Ide and install
-streamlit
-pickle
-pandas

Step 4:
Run app.py

And Your Recommendation System is ready.
